Работа с рассказами и главами

  1. Нажмите ссылку «Добавить рассказ» на панели навигации.
  2. Правильно заполните все необходимые поля.
  3. Нажмите кнопку «Добавить рассказ» внизу формы.

Таким образом добавляется шапка будущего рассказа. Пока в нем нет глав, его нельзя посмотреть или опубликовать.

Следующий шаг – добавление глав.


Если добавляется новый рассказ, то пункты 1 и 2 можно пропустить – после добавления идёт автоматическое перенаправление на страницу редактирования.

  1. Нажмите ссылку «Кабинет автора» на панели навигации.
  2. Нажмите кнопку «Редактировать» для нужного рассказа.
  3. Измените необходимые параметры рассказа и/или отсортируйте главы перетаскиванием.
  4. Нажмите кнопку «Сохранить изменения» внизу формы.

  1. Откройте рассказ для редактирования.
  2. Нажмите кнопку «Добавить главу» внизу формы.
  3. Правильно заполните все необходимые поля.
  4. Нажмите кнопку «Добавить» внизу формы.

Таким образом добавляется новая глава.

После добавления идёт перенаправление на страницу редактирования главы, где можно увидеть, как выглядит текст.


  1. Откройте рассказ для редактирования.
  2. Нажмите ссылку «править» для нужной главы.
  3. Правильно заполните все необходимые поля.
  4. Нажмите соответствующую кнопку внизу формы:
    • «Сохранить изменения» – для того, чтобы сохранить главу;
    • «Удалить главу» – для ее удаления (нужно подтверждение).

  1. Нажмите ссылку «Кабинет автора» на панели навигации.
  2. Нажмите соответствующую кнопку возле рассказа.
    • «Редактировать» – для правки рассказа;
    • «Опубликовать»/«В черновики» – для изменения опубликованности. Первое нажатие отправляет рассказ на рассмотрение;
    • «Удалить» – для удаления рассказа (нужно подтверждение).
